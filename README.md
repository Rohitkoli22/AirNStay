# AirNStay
AirNStay is a full-stack Travel and Hospitality Management System designed to handle hotel bookings, flight reservations, and travel packages through a role-based architecture. The platform is being developed with a scalable, enterprise-ready approach using Angular for the frontend and Spring Boot for the backend.

# Current Status

Frontend developed using Angular CLI 15

Backend integration with Spring Boot is in progress

Mock APIs implemented using db.json

# User Roles

Traveler – Book hotels & flights, manage profile, payments, and support

Travel Agent – Manage flight details (planned)

Hotel Agent – Manage hotel listings (planned)

Package Agent – Manage travel packages (in progress)

Admin – Role & system management (planned)

# Features Implemented

Hotel Booking

Flight Booking

Support (FAQs, Tickets, Chatbot UI)

Profile Management

Payment Module

Role-based Dashboard

Mock API integration (db.json)

# Tech Stack

Frontend: Angular 15, TypeScript
Backend: Java Spring Boot (in progress)
Data: JSON Server (db.json)
